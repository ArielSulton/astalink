## Backend

### New Endpoints

| Method | Path | Auth | Description |
|--------|------|------|-------------|
| `GET` | `/api/v1/market/news` | No | Fetch sentiment-tagged news articles by ticker |
| `GET` | `/api/v1/legal/documents` | No | List indexed regulation documents |
| `POST` | `/api/v1/legal/documents/upload` | JWT | Upload PDF to index (max 10 MB) |

### Bug Fixes

- **`checkpointer.py`**: `PostgresSaver.from_conn_string()` returns a context manager in `langgraph-checkpoint-postgres >= 2.x` — fixed by calling `.__enter__()` to get the actual saver instance; falls back to `MemorySaver` on failure
- **Thread scoping**: Chat `thread_id` scoped to `{user_sub}:{raw_thread}` to prevent cross-user history access
- **`market.py`**: Removed auth requirement from public watchlist endpoint; wrapped sync `fetch_news` with `asyncio.to_thread`

## Known Limitations

- `NEWS_API_KEY` is optional; news page shows empty state when unset
- `GOOGLE_API_KEY` / `PINECONE_API_KEY` required for full AI narration and legal RAG — placeholders in `.env.example`
- Chatbot thread history does not survive if `MemorySaver` fallback is active (requires `SUPABASE_DB_URL`)
- Direct Postgres connection (`port 5432`) may be firewalled in some environments; use the container's pooler URL for migrations

## Upgrade Notes

If upgrading from `v0.1.x`, apply the `regulation_documents` migration manually or via drizzle-kit from inside the Docker container:

```bash
docker exec astalink-backend-1 uv run python -c "
import psycopg, os
conn_str = os.environ['SUPABASE_DB_URL']
sql = '''
CREATE TABLE IF NOT EXISTS public.regulation_documents (
    id uuid PRIMARY KEY DEFAULT gen_random_uuid(),
    source text NOT NULL, title text NOT NULL, version text,
    doc_hash text NOT NULL UNIQUE,
    indexed_at timestamptz NOT NULL DEFAULT now(),
    metadata jsonb NOT NULL DEFAULT \'{}\'::jsonb
);
ALTER TABLE public.regulation_documents ENABLE ROW LEVEL SECURITY;
CREATE POLICY IF NOT EXISTS regulation_documents_select_all
    ON public.regulation_documents FOR SELECT USING (true);
'''
with psycopg.connect(conn_str, autocommit=True) as c:
    c.execute(sql)
    print('done')
"
```
